Language Guide: Collection Types

Oct 3rd, 2014 2:37 pm

禁止用于商业用途，转载请注明出处

Swift提供了两种容器类型，数组和字典。数组相同类型对象的有序列表。字典存储相同类型的无序的值的组合，每个值都可以通过一个标示符（键）来查询。

Swift里数组和字典对他们所包含的值的类型是知道的。这意味着，你不能插入不同类型的值。这也意味着，当你从他们中取值时，取到的值的类型也是确定的。

注意
数组和字典底层是通过泛型容器来实现的。
